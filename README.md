# ntumaker_20201010

### 組別:36 吃飽睡睡飽吃

### Description
我們希望透過這個作品喚起人們的環保意識，在都市化的社會裡，透過與植物互動的方式拉近人與自然的距離。

### Usage
以直播方式在TWITCH平台展示，當有人捐款或追蹤就會有不同的效果展示，根據金額大小，分別有以下功能。

| 捐款金額(美元) | 功能                                 |
|----------------|--------------------------------------|
| 0~10           | 澆水3秒                              |
| 10~50          | 澆水5秒                              |
| 50~100         | 施肥2秒                              |
| >100           | 澆水5秒+施肥2秒+LED燈光效果+動態展示 |
| 追蹤           | LED燈光效果動態展示                  |
